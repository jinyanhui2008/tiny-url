# tiny-url

#### 介绍
也许是性能最高的短网址服务。

#### 软件架构
基于spring boot开发, 结合nginx高效的静态资源访问效率, 提供高效的短网址服务.


#### 安装教程

1.  安装nginx程序.  
2.  修改application.yml配置文件,修改tiny.file.parh路径为文件存储路径  
3.  通过maven编译程序
4.  部署编译号的jar程序到本地
